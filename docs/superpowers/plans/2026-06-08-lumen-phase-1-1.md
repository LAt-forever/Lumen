# Lumen Phase 1.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close Phase 1 gaps by adding confirmed memory management, file/link source capture, real nav views, and backend lifecycle cleanup.

**Architecture:** Extend existing FastAPI service/repository boundaries without schema migrations. Normalize uploaded files and captured links into the existing `sources.content` field so current indexing and citation behavior continue to work. Extend the React workbench with internal view state, reusing TanStack Query hooks and existing panel styling.

**Tech Stack:** Python 3.12+, FastAPI, SQLAlchemy 2, pytest, httpx, pypdf, BeautifulSoup, React 18, TypeScript, Vite, TanStack Query, Vitest, Testing Library, lucide-react.
